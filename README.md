# DonApp: aplicativo para gestão de doações

Este é o projeto de um sistema para cadastrar doações e donatários.
O projeto está codificado em python.

Trata-se de um projeto educativo, limitado ao caso de doação de roupas.

## Versão 
Python 3.11.9

## Instalação
1. Clonar o repositório:
```bash
git clone https://github.com/eduardo-kt/donap.git
cd donap
```
2. Criar ambiente virtual
```bash
python -m venv .venv
```
3. Ativar ambiente virtual
   1. No Windows
```bash
.\.venv\Scripts\activate
```
   2. No Linux/Mac
```bash  
source .venv/bin/activate
```
4. Instalar dependências
```bash
pip install -r requirements.txt
```
